## Technical SEO

### Technical SEO Audit Checklist

**Prompt:**
```
Create a comprehensive technical SEO audit checklist for [WEBSITE URL or TYPE].

Website type: [E-COMMERCE/BLOG/SERVICE SITE/SAAS]
Platform: [WORDPRESS/SHOPIFY/CUSTOM/OTHER]
Size: [APPROXIMATE NUMBER OF PAGES]

Generate a prioritized checklist covering:

1. CRAWLABILITY & INDEXABILITY
   - [ ] Robots.txt properly configured
   - [ ] XML sitemap exists and submitted
   - [ ] No orphaned pages
   - [ ] Canonical tags implemented correctly
   - [ ] No crawl errors in GSC
   - [ ] JavaScript rendering check
   - [ ] Pagination handling
   - [ ] Noindex tags audit

2. SITE SPEED & PERFORMANCE
   - [ ] Core Web Vitals scores (LCP, FID, CLS)
   - [ ] Page load time under 3 seconds
   - [ ] Image optimization
   - [ ] Minimize CSS/JavaScript
   - [ ] Browser caching enabled
   - [ ] CDN implementation
   - [ ] Server response time
   - [ ] Mobile performance

3. MOBILE OPTIMIZATION
   - [ ] Mobile-friendly test passed
   - [ ] Responsive design implementation
   - [ ] Touch elements properly sized
   - [ ] No horizontal scrolling
   - [ ] Mobile page speed
   - [ ] Mobile usability issues

4. SITE ARCHITECTURE
   - [ ] Clear URL structure
   - [ ] Logical site hierarchy (3-click rule)
   - [ ] Internal linking strategy
   - [ ] Breadcrumb navigation
   - [ ] 404 error pages
   - [ ] Redirect chains audit
   - [ ] Site search functionality

5. ON-PAGE ELEMENTS
   - [ ] Title tags (all pages)
   - [ ] Meta descriptions (all pages)
   - [ ] Header tag hierarchy (H1-H6)
   - [ ] Image alt text
   - [ ] Schema markup
   - [ ] Open Graph tags
   - [ ] Twitter Cards

6. SECURITY & ACCESSIBILITY
   - [ ] HTTPS implementation
   - [ ] SSL certificate valid
   - [ ] Mixed content issues
   - [ ] WCAG compliance level
   - [ ] Alt text for accessibility
   - [ ] Keyboard navigation
   - [ ] Color contrast ratios

7. INTERNATIONAL SEO (if applicable)
   - [ ] Hreflang tags
   - [ ] Language targeting
   - [ ] Geographic targeting
   - [ ] Duplicate content across regions

For each item, include:
- Why it matters
- How to check it
```
