## üìã Layout Structure (Top to Bottom)

### 1. Top Section (0-15%)
- **Background:** Gradient from Dark Navy (#0f172a) to Slate Blue (#1e293b)
- **Accent bar:** 8px horizontal bar at very top with blue-to-purple gradient
- **Decorative circle:** Semi-transparent dark circle in top-left corner (opacity 30%)

### 2. Badge Section (15-20%)
- **"AI POWERED" badge:** Rounded rectangle (pill shape)
  - Width: 400px, Height: 120px
  - Border radius: 60px
  - Fill: Blue-to-purple gradient (left to right)
  - Text: White, centered, bold, all caps

### 3. Main Title Section (25-45%)
- **Title text:** Center-aligned, three lines:
  - Line 1: "AI MARKETING &"
  - Line 2: "SEO PROMPT"
  - Line 3: "PACK"
- **Accent line:** 300px wide, 4px height, centered, blue-to-purple gradient, 30px below title

### 4. Subtitle Section (50-58%)
- **Subtitle:** Two lines, center-aligned:
  - Line 1: "High-Conversion AI Prompts"
  - Line 2: "for Marketers & Agencies"
- Spacing: 50px from accent line

### 5. Features Section (62-80%)
- **Feature list:** Left-aligned with bullet points
  - Bullet style: Small circles (12px diameter)
  - Alternate colors: Blue, Purple, Blue, Purple
  - Text spacing: 80px between each line
  - Indent: 70px from bullet to text
  
**Feature bullets:**
1. "105+ Battle-Tested Prompts"
2. "SEO, Ads, Email & Landing Pages"
3. "Proven Marketing Frameworks"
4. "Works with ChatGPT, Claude & More"

### 6. Decorative Elements (78-82%)
- **Dot pattern:** Small circles (8px) in a row
  - Pattern: Blue, Blue, Purple, Purple, [gap], Purple, Purple, Blue, Blue
  - Opacity: 15%
  - Purpose: Visual separator

### 7. Bottom Section (85-100%)
- **Background overlay:** Semi-transparent dark navy (50% opacity)
- **Branding text:** Center-aligned:
  - "CREATED BY" (small, all caps, gray)
  - "MHM IT" (large, bold, white)
  - "Professional Marketing Solutions" (small, dark gray)

## üé≠ Design Style Guidelines

### Overall Aesthetic
- **Style:** Modern, professional, tech-forward
- **Mood:** Trustworthy, premium, innovative
- **Approach:** Clean, minimal, not cluttered

### Visual Elements
- **Shapes:** Geometric, rounded corners (modern feel)
- **Gradients:** Subtle, left-to-right or top-to-bottom
- **Shadows:** Minimal or none (flat design)
- **Spacing:** Generous whitespace, breathing room

### Do's
‚úÖ Use sharp, high-contrast text  
‚úÖ Keep gradients subtle and professional  
‚úÖ Maintain consistent spacing  
‚úÖ Use geometric shapes sparingly  
‚úÖ Keep the design clean and modern  

### Don'ts
‚ùå Don't overcrowd with elements  
‚ùå Don't use more than 2-3 colors  
‚ùå Don't use decorative or script fonts  
‚ùå Don't add stock photos or complex imagery  
‚ùå Don't use thin or light font weights (poor readability)

## üõ†Ô∏è Step-by-Step Recreation in Canva

### Step 1: Setup Canvas
1. Create new design: Custom size 1600 x 2400 px
2. Set background: Gradient from #0f172a (top) to #1e293b (bottom)

### Step 2: Add Top Accent Bar
1. Add rectangle at top: 1600 x 16 px
2. Fill: Gradient left-to-right, #3b82f6 to #8b5cf6

### Step 3: Create "AI POWERED" Badge
1. Add rounded rectangle: 400 x 120 px, position at top center (y: 240px)
2. Border radius: Maximum (pill shape)
3. Fill: Gradient #3b82f6 to #8b5cf6
4. Add text: "AI POWERED", white, bold, 56px, centered

### Step 4: Main Title
1. Add text box centered on canvas
2. Type: "AI MARKETING &\nSEO PROMPT\nPACK"
3. Font: Inter Bold or Poppins Bold
4. Size: 104px
5. Color: White
6. Alignment: Center
7. Position: y: 600px (approximately)

### Step 5: Accent Line
1. Add rectangle: 300 x 4 px
2. Border radius: 2px
3. Fill: Gradient #3b82f6 to #8b5cf6
4. Position: Below title, centered

### Step 6: Subtitle
1. Add text: "High-Conversion AI Prompts\nfor Marketers & Agencies"
2. Font: Inter Regular, 48px
3. Color: #94a3b8
4. Center aligned
5. Position: ~100px below accent line

### Step 7: Feature Bullets
1. Add 4 small circles (12px each)
2. Color: Alternate #3b82f6 and #8b5cf6
3. Add text for each feature
4. Font: Inter Regular, 36px, color #cbd5e1
5. Left-align as a group, center the group on canvas

### Step 8: Decorative Dots
1. Add 8 small circles (8px)
2. Arrange in row pattern described above
3. Set opacity: 15%
4. Position: Between features and bottom section

### Step 9: Bottom Section
1. Add rectangle: 1600 x 300 px at bottom
2. Fill: #0f172a with 50% opacity
3. Add three text elements:
   - "CREATED BY" - Inter Regular, 40px, #94a3b8, all caps
   - "MHM IT" - Inter Bold, 64px, white
   - "Professional Marketing Solutions" - Inter Regular, 32px, #64748b
4. Stack vertically, center aligned

### Step 10: Decorative Circles (Optional)
1. Add large circle in top-left: 300px diameter
2. Fill: #1e293b, opacity 30%
3. Position: Partially off-canvas (top-left corner)
4. Add large circle in bottom-right: 400px diameter
5. Fill: #1e293b, opacity 30%
6. Position: Partially off-canvas (bottom-right corner)

### Step 11: Export
1. Download as PNG (high quality)
2. Or download as PDF for print quality
